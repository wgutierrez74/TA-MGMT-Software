TA Management Software

This is a system for Prospective TAs and Faculty which allows students who would like to be TAs to apply for the classes they would most like to TA. It also allows Faculty to recommend students, and hopefully find a perfect fit for both parties.

This software is to help Prospective CISE TAs choose the classes they would like to TA for.
